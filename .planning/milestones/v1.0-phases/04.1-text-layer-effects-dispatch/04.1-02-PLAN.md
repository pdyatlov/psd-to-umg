**lfx2 block layout (verified in 04.1-RESEARCH.md §Pattern 2):**
- TaggedBlock with `getKey() == NAMESPACE_PSAPI::Enum::TaggedBlockKey::Unknown` AND `Block->m_Data` prefix matches descriptor version marker `0x00 0x00 0x00 0x10` (big-endian uint32 = 16).
- After the version prefix: a Photoshop Descriptor (recursive key-value tree). The outer descriptor has class-id `"null"` and contains a key `"FrFX"` whose value is itself a descriptor describing the stroke:
  - `"enab"` → bool (enabled flag)
  - `"Styl"` → enum (position: InsetFrame/OutsetFrame/CenterFrame — ignored this phase, D-12 note)
  - `"PntT"` → enum (fill type: `SClr`=solid, others ignored)
  - `"Sz  "` → UntF (size, Unit=`#Pxl`, value=double, PSD pixels)  ← note two trailing spaces in key
  - `"Clr "` → Descriptor (RGBC color sub-descriptor)  ← note one trailing space
  - `"Opct"` → UntF (opacity, Unit=`#Prc`, value=0..100 percent double)

**Photoshop Descriptor binary format (simplified; see 04.1-RESEARCH.md for full spec):**
```
Descriptor:
  [unicode_string] class_id_name   (uint32 length, then length*2 bytes UTF-16 BE)
  [ps_string]      class_id        (uint32 length; if length==0, read 4 bytes as class_id)
  [uint32]         item_count
  for each item:
    [ps_string]    key
    [4 bytes]      ostype            ('Objc','bool','doub','UntF','long','enum','TEXT','tdta','VlLs', ...)
    [data]         payload (varies by ostype)

ostype 'bool' : [1 byte]  = 0/1
ostype 'long' : [int32]   big-endian
ostype 'doub' : [double]  big-endian 8 bytes
ostype 'UntF' : [4 bytes unit code] [double value]
ostype 'enum' : [ps_string type] [ps_string value]
ostype 'Objc' : nested Descriptor (recursive)
ostype 'TEXT' : [unicode_string]
ostype 'tdta' : [uint32 len][bytes]     // skip
ostype 'VlLs' : [uint32 n] then n entries [ostype][payload]
ostype 'UnFl' : [uint32 unit][uint32 n][doubles...]

ps_string (aka 4-char-or-padded key): [uint32 len]; if len==0, read 4 bytes; else read len bytes ASCII.
```

**Color sub-descriptor (`"Clr "` value):** Objc with class_id `"RGBC"` and three doub items keyed `"Rd  "` / `"Grn "` / `"Bl  "` (trailing-space keys), each a `doub` in range 0..255. No alpha; compose alpha from `"Opct"` percentage / 100.

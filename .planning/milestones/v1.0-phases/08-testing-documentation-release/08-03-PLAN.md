Structure (per research recommendations):

1. **# PSD2UMG** — One-line description. Badges: `UE 5.7` `Win64` `MIT License`. Description: "Unreal Engine 5.7 editor plugin that imports .psd files and converts them into UMG Widget Blueprints. Drop a PSD, get a working UI."

2. **## Quick Start** — 4 numbered steps:
   1. Copy `PSD2UMG/` folder to your project's `Plugins/` directory
   2. Regenerate project files (right-click .uproject > Generate Visual Studio project files)
   3. Open UE editor, enable PSD2UMG in Edit > Plugins if not auto-enabled
   4. Drag a `.psd` file into Content Browser — a Widget Blueprint is created automatically

3. **## Installation** — Detailed:
   - Requirements: UE 5.7+, Win64, Visual Studio 2022+
   - Clone or download, copy to Plugins/
   - Add to .uproject Plugins array: `{"Name": "PSD2UMG", "Enabled": true}`
   - Build from source (if needed)
   - Optional plugins: CommonUI (for UCommonButtonBase), EnhancedInput (for input action binding)

4. **## Layer Naming Cheat Sheet** — THE core section. Two tables.

   **Prefixes** (source from AllMappers.h registry — executor must read that file to get exact list):
   | Prefix | Widget | Notes |
   |--------|--------|-------|
   | Button_ | UButton | Normal/Hovered/Pressed/Disabled child layers for state brushes |
   | Progress_ | UProgressBar | Background/Fill child layers |
   | HBox_ | UHorizontalBox | Children laid out left-to-right |
   | VBox_ | UVerticalBox | Children laid out top-to-bottom |
   | Overlay_ | UOverlay | Z-stacked children |
   | ScrollBox_ | UScrollBox | Scrollable container |
   | Slider_ | USlider | |
   | CheckBox_ | UCheckBox | |
   | Input_ | UEditableTextBox | |
   | List_ | UListView | |
   | Tile_ | UTileView | |
   | Switcher_ | UWidgetSwitcher | |

   **Suffixes** (source from FAnchorCalculator.cpp GSuffixes table):
   | Suffix | Effect |
   |--------|--------|
   | _9s or _9slice | 9-slice (Box draw mode). Optional margins: `_9s[16,16,16,16]` |
   | _variants | Widget Switcher (one child per variant) |
   | _anchor-tl | Force top-left anchor |
   | _anchor-tr | Force top-right anchor |
   | _anchor-bl | Force bottom-left anchor |
   | _anchor-br | Force bottom-right anchor |
   | _anchor-c | Force center anchor |
   | _stretch-h | Stretch horizontally |
   | _stretch-v | Stretch vertically |
   | _fill | Fill entire parent |

   **CommonUI mode** (when enabled in settings):
   | Syntax | Effect |
   |--------|--------|
   | Button_Name | UCommonButtonBase instead of UButton |
   | Button_Confirm[IA_Confirm] | Bind to UInputAction asset |
   | _show / _hide / _hover | Generate UWidgetAnimation |

5. **## Plugin Settings** — Project Settings > Plugins > PSD2UMG. List each field from UPSD2UMGSettings (executor reads PSD2UMGSetting.h):
   - TargetDir (output directory for Widget Blueprints)
   - TextureDir (texture output subdirectory)
   - FontMap (TMap of PS font name to UFont asset)
   - SourceDPI (default 72)
   - bFlattenComplexEffects (rasterize unsupported effects)
   - bAutoAssignAnchors (heuristic anchor placement)
   - bUseCommonUI (CommonUI button mode)
   - DefaultFontAsset (fallback UFont)

6. **## Running Tests** — Three methods:
   - Session Frontend: Window > Developer Tools > Session Frontend > Automation > filter "PSD2UMG" > Run
   - Command line: `UnrealEditor.exe MyProject.uproject -ExecCmds="Automation RunTests PSD2UMG;Quit" -log -nosplash -nullrhi`
   - Filter by subsystem: PSD2UMG.Parser.*, PSD2UMG.Generator.*, PSD2UMG.Typography.*

7. **## Test Fixtures** — Brief note: test PSDs live in `Source/PSD2UMG/Tests/Fixtures/`. List the 5 fixtures (MultiLayer.psd, Typography.psd, SimpleHUD.psd, ComplexMenu.psd, Effects.psd). Can also be dragged into Content Browser for manual testing.

8. **## Known Limitations** — Bullet list:
   - Win64 only (macOS support deferred)
   - UE 5.7+ required
   - Effects fidelity is approximate (color overlay, drop shadow, opacity); complex effects flatten to PNG
   - Text shadow not supported (PhotoshopAPI limitation)
   - Smart Object recursive import has depth limit
   - No runtime module (editor-only)

9. **## License** — MIT. One line.

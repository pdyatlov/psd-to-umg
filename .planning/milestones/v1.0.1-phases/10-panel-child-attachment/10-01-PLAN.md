    3. Inside the loop (line ~88), replace the direct `UCanvasPanelSlot* Slot = Parent->AddChildToCanvas(Widget);` call with the dispatch pattern:
       ```cpp
       UPanelSlot* Slot = nullptr;
       UCanvasPanel* CanvasParent = Cast<UCanvasPanel>(Parent);
       if (CanvasParent)
       {
           Slot = CanvasParent->AddChildToCanvas(Widget);
       }
       else
       {
           Slot = Parent->AddChild(Widget);
       }
       if (!Slot) { continue; }
       ```

    6. FX-01 (opacity), FX-02 (visibility), FX-03 (color overlay) at lines ~184-209 operate on `Widget` directly, not on the slot — they must run regardless of parent panel type. Move them outside the canvas-slot guard but keep them inside the `if (Slot != nullptr)` region.

           TEXT("Drop shadow on layer '%s' inside non-canvas parent '%s' — no-op (canvas-only sibling pattern)."),

       The existing warning at line ~247 (drop shadow on non-image layer) remains unchanged.

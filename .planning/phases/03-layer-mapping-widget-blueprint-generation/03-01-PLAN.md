**FTextureImporter.h**:
```cpp
#pragma once
#include "CoreMinimal.h"

struct FPsdLayer;
class UTexture2D;

class PSD2UMG_API FTextureImporter
{
public:
    // Import a single layer's RGBA pixels as a persistent UTexture2D.
    // PackageBasePath = e.g. "/Game/UI/Textures/MyHUD"
    // Returns nullptr on skip (empty pixels, zero bounds).
    static UTexture2D* ImportLayer(const FPsdLayer& Layer, const FString& PackageBasePath);

    // Build the base path from settings: {TextureAssetDir}/Textures/{PsdName}
    static FString BuildTexturePath(const FString& PsdName);

    // Deduplicate asset names: append _01, _02 if name already used
    static FString DeduplicateName(const FString& BaseName, const TSet<FString>& UsedNames);
};
```

**FTextureImporter.cpp**:
1. `ImportLayer`: 
   - Check `Layer.RGBAPixels.Num() == 0 || Layer.PixelWidth <= 0 || Layer.PixelHeight <= 0` → return nullptr with `UE_LOG(LogPSD2UMG, Warning, ...)` per D-14
   - Copy `Layer.RGBAPixels` to `BGRAPixels` array
   - Swap R and B channels: `for (int32 i = 0; i < BGRAPixels.Num(); i += 4) { Swap(BGRAPixels[i], BGRAPixels[i+2]); }`
   - `CreatePackage` with path `{PackageBasePath}/{AssetName}`
   - `FullyLoad()` on the package
   - `NewObject<UTexture2D>(Pkg, FName(*AssetName), RF_Public | RF_Standalone)`
   - `Tex->PreEditChange(nullptr)`
   - `Tex->Source.Init(Layer.PixelWidth, Layer.PixelHeight, 1, 1, TSF_BGRA8, BGRAPixels.GetData())`
   - `Tex->SRGB = true; Tex->CompressionSettings = TC_Default;`
   - `Tex->PostEditChange()`
   - `Pkg->MarkPackageDirty()`
   - `UEditorAssetLibrary::SaveLoadedAsset(Tex, false)`
   - Return Tex

2. `BuildTexturePath`: Read `UPSD2UMGSettings::Get()->TextureAssetDir.Path`. If empty, default to `/Game/UI`. Return `{Dir}/Textures/{PsdName}`.

3. `DeduplicateName`: If BaseName not in UsedNames, return it. Otherwise try `BaseName_01`, `BaseName_02`, ... up to `_99`.

Include headers: `Engine/Texture2D.h`, `UObject/UObjectGlobals.h` (CreatePackage), `EditorAssetLibrary.h`, `PSD2UMGLog.h`, `PSD2UMGSetting.h`, `Parser/PsdTypes.h`.

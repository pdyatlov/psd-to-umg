- `Map`:

**FButtonLayerMapper.cpp** (Priority: 200 — higher than Group so prefix wins):
- `CanMap`: `Layer.Type == EPsdLayerType::Group && Layer.Name.StartsWith(TEXT("Button_"))`
- `Map`:
  - `UButton* Btn = Tree->ConstructWidget<UButton>(UButton::StaticClass(), FName(*Layer.Name))`
  - `FButtonStyle Style = FButtonStyle::GetDefault();`
  - Build texture base path: `FTextureImporter::BuildTexturePath(FPaths::GetBaseFilename(Doc.SourcePath))`
  - Iterate `Layer.Children` (image-type children only):
    - Import each child as texture via `FTextureImporter::ImportLayer`
    - Match child name suffix to state (case-insensitive):
      - `_Normal`, `_Bg`, or first image child (index 0 fallback) → `Style.SetNormal(Brush)`
      - `_Hovered`, `_Hover` → `Style.SetHovered(Brush)`
      - `_Pressed`, `_Press` → `Style.SetPressed(Brush)`
      - `_Disabled` → `Style.SetDisabled(Brush)`
    - Build `FSlateBrush`: `Brush.SetResourceObject(Tex); Brush.ImageSize = FVector2D(Child.PixelWidth, Child.PixelHeight);`
  - `Btn->SetStyle(Style)` — use SetStyle, NOT direct WidgetStyle access (deprecated 5.2)
  - Return Btn

**FProgressLayerMapper.cpp** (Priority: 200):
- `CanMap`: `Layer.Type == EPsdLayerType::Group && Layer.Name.StartsWith(TEXT("Progress_"))`
- `Map`:
  - `UProgressBar* Bar = Tree->ConstructWidget<UProgressBar>(UProgressBar::StaticClass(), FName(*Layer.Name))`
  - `FProgressBarStyle Style = FProgressBarStyle::GetDefault();`
  - Iterate `Layer.Children`:
    - `_BG`, `_Background` → `Style.BackgroundImage = Brush`
    - `_Fill`, `_Foreground` → `Style.FillImage = Brush`
  - `Bar->SetWidgetStyle(Style)` — use SetWidgetStyle, NOT direct field access
  - Return Bar

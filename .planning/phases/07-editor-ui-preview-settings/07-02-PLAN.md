Implement the full Slate dialog per UI-SPEC layout:

**Construct():** Build SVerticalBox with 3 slots:

Slot A — Layer Tree (FillHeight 1.0):
- SBorder with FAppStyle::GetBrush("ToolPanel.GroupBorder")
- SScrollBox containing STreeView<TSharedPtr<FPsdLayerTreeItem>>
- Tree delegates: OnGenerateRow, OnGetChildren
- Expand all items on construct via TreeView->SetItemExpansion(item, true) recursively

Slot B — Output Path Row (AutoHeight, 36px):
- SHorizontalBox: label "Output Path:" (88px), SEditableTextBox (fill), browse SButton (28x28 with FAppStyle::GetBrush("Icons.FolderOpen"))
- Padding FMargin(8, 4)
- Below the path row: a conditional STextBlock for validation error "Path must start with /Game/ -- enter a valid content path." in red FLinearColor(0.9f, 0.2f, 0.2f, 1.f), visibility bound to path validity

Slot C — Button Bar (AutoHeight, 40px):
- SBorder with "ToolPanel.GroupBorder"
- SHorizontalBox: fill spacer, Cancel SButton, Import/Apply Reimport SButton
- Import button label: bIsReimport ? "Apply Reimport" : "Import"

**OnGenerateRow():** Per UI-SPEC STreeView Row spec:
- STableRow containing SHorizontalBox with 5 slots:
  - Col 1: SCheckBox (AutoWidth, Padding 4,0)
  - Col 2: SSpacer (8px * Item->Depth wide)
  - Col 3: STextBlock layer name (FillWidth 1.0, NormalFont)
  - Col 4: Badge — SBorder with FMargin(4,2), badge color background, containing STextBlock (SmallFont, white, widget type name), fixed 120px
  - Col 5: Change annotation STextBlock (AutoWidth, 80px) — visible only when bIsReimport. Color: [new]=green(0.3,0.75,0.3), [changed]=amber(0.9,0.65,0.1), [unchanged]=DefaultForeground at 50% alpha

**Checkbox cascade (per UI-SPEC Interaction Contracts):**
- OnCheckStateChanged: set Item->bChecked, then call SetChildrenChecked recursively
- GetCheckState: if Item has children and some checked/some not, return Undetermined
- Parent re-check restores all children to checked

**BuildTreeFromDocument():** Recursively walk FPsdDocument::RootLayers. For each FPsdLayer:
- Create TSharedPtr<FPsdLayerTreeItem>
- Use FLayerMappingRegistry::Get().FindMapper(Layer) to get widget type name; if nullptr, use "CanvasPanel" for groups, "Unknown" otherwise
- Badge colors per UI-SPEC color table:
  - Contains "Button": FLinearColor(0.2f, 0.5f, 0.9f, 1.f)
  - Contains "Image": FLinearColor(0.3f, 0.65f, 0.3f, 1.f)
  - Contains "TextBlock": FLinearColor(0.8f, 0.6f, 0.1f, 1.f)
  - Contains "CanvasPanel": FLinearColor(0.5f, 0.5f, 0.5f, 1.f)
  - Otherwise: FLinearColor(0.4f, 0.4f, 0.6f, 1.f)

**OnBrowseClicked():** Use IDesktopPlatform::OpenDirectoryDialog, write result to OutputPathBox.

**OnImportClicked():** Validate path starts with "/Game/". If invalid, show error text. If valid, fire OnConfirmed delegate with path and RootItems, then return FReply::Handled().

**OnCancelClicked():** Fire OnCancelled delegate, return FReply::Handled().
